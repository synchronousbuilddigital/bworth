## Contact Form Email Setup

1. Copy `.env.example` to `.env.local`.
2. Configure SMTP values (`SMTP_HOST`, `SMTP_PORT`, `SMTP_SECURE`, `SMTP_USER`, `SMTP_PASS`).
3. Set `CONTACT_RECEIVER_EMAILS` as comma-separated addresses where contact form emails should go.
4. Optional: set `CONTACT_CC_EMAILS` for additional recipients.

The contact form sends:
- a notification email to all configured receiver emails,
- and an auto-reply email to the user who submitted the form.

Notification emails include `reply-to` as the user email so your team can reply directly.
